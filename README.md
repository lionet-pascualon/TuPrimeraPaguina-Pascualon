# CODINAPP - Gestión Académica 🚀

Este proyecto es una plataforma de gestión para instituciones educativas, desarrollada como parte del curso de Python en Coderhouse.

## 📋 Funcionalidades
* **Cursos:** Administración de comisiones y nombres de cursos.
* **Profesores:** Registro de docentes (Nombre, Apellido, Email y Profesión).
* **Estudiantes:** Gestión de alumnos inscriptos.
* **Buscador:** Filtro dinámico para encontrar cursos por número de camada.

## 🛠️ Tecnologías utilizadas
* **Python 3.x**
* **Django 5.x**
* **Bootstrap 5** (Para el diseño visual)
* **SQLite3** (Base de datos)

## 🚀 Cómo ejecutar el proyecto
1. Clonar el repositorio.
2. Instalar Django si no lo tienes: `pip install django`.
