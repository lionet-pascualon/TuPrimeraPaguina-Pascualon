# CODINAPP - Proyecto Final Django

Este es un sistema de gestión académica desarrollado para el curso de Python de Coderhouse.

## Funcionalidades
* **Cursos:** Visualización y carga de comisiones.
* **Profesores:** Gestión de docentes (Nombre, Profesión, Email).
* **Estudiantes:** Registro de alumnos.
* **Buscador:** Búsqueda dinámica de cursos por número de camada.

## Cómo ejecutarlo
1. Clonar el repositorio.
2. Ejecutar `python manage.py migrate`.
3. Ejecutar `python manage.py runserver`.
